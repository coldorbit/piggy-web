# ApplyPilot Client

React/Vite client for ApplyPilot.

## Setup

```bash
cp .env.example .env
pnpm install
pnpm dev
```

Default client URL: `http://localhost:3000`.

The client expects the API at `VITE_API_BASE_URL`, defaulting to relative API paths when unset.

## Scripts

```bash
pnpm dev
pnpm build
pnpm preview
pnpm check
```
